# implement-page

Implement or update a Weavers page.

Page: [PAGE_NAME]

Requirements:
1. Use the approved Stitch design where available.
2. Follow Weavers brand and design rules.
3. Use reusable components.
4. Add correct copy.
5. Add responsive mobile layout.
6. Add CTA routing.
7. Add SEO metadata.
8. Add accessibility basics.
9. Add analytics hooks if available.
10. Run available checks.

Return:
- Files changed
- Components created/updated
- Checks run
- Manual QA steps

This command will be available in chat with /implement-page
