# Premium Page Builder Skill

Use this skill when creating a complete Weavers page.

Required page ingredients:
1. Purpose-driven hero
2. Strong primary CTA
3. Secondary CTA where useful
4. Product or service explanation
5. Trust elements
6. Mobile-first layout
7. SEO metadata
8. Accessible headings
9. Clear next step

Page standards by type:

Home:
- Brand promise
- Product pillars
- Personalization
- Denim feature
- Weddingwear feature
- Instagram Looks
- Fit confidence
- Final CTA

Product pages:
- Explain category clearly.
- Show customization options.
- Include measurement path.
- Include reference upload path.
- Include CTA.

Weddingwear:
- Consultation-first.
- Capture event date and need-by date.
- Include role and number of outfits.
- Do not force instant checkout.

Instagram Looks:
- Lookbook cards.
- Filters.
- Recreate flow.
- Reference upload.

Fit Guide:
- Measurement methods.
- Unit toggle.
- Confidence selector.
- Video fitting CTA.

Acceptance checklist:
- Looks premium.
- Works on mobile.
- Has CTA.
- Has metadata.
- Has no dead buttons.
- Does not drift from brand positioning.
